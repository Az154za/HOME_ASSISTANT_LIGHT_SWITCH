# HOME_ASSISTANT_LIGHT_SWITCH
# Description : 
  This project is made to automate , optimize and monitor house temperature , motion , toggle on and off lights via two servo motors , DHT11 for temperature sensor and motion sensor light toggling automatic with home assistant os
